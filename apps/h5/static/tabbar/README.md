# TabBar 图标说明

> **注意**: 由于无法提供真实的 PNG 图标文件，当前使用占位方案。请设计师提供以下图标后替换。

## 图标规格要求

| 属性     | 要求                                                      |
| -------- | --------------------------------------------------------- |
| 格式     | PNG (透明背景)                                            |
| 尺寸     | 81×81 px (@3x)                                            |
| 颜色模式 | 默认: `#64748B` (slate-500) / 选中: `#F59E0B` (amber-500) |
| 风格     | 线性图标 (line-style)，2px 描边                           |

## 所需图标清单

### 1. 首页 (home)

- **默认** (`home.png`): 房子轮廓图标，颜色 `#64748B`
- **选中** (`home-active.png`): 房子轮廓图标，颜色 `#F59E0B`
- **设计参考**: 首页/主页的经典房子形状

### 2. 发现 (discover)

- **默认** (`discover.png`): 探索/罗盘图标，颜色 `#64748B`
- **选中** (`discover-active.png`): 探索/罗盘图标，颜色 `#F59E0B`
- **设计参考**: 圆形罗盘或放大镜+星星组合

### 3. 服务 (services)

- **默认** (`services.png`): 方块网格/服务列表图标，颜色 `#64748B`
- **选中** (`services-active.png`): 方块网格/服务列表图标，颜色 `#F59E0B`
- **设计参考**: 四宫格或列表样式图标

### 4. AI 助手 (ai)

- **默认** (`ai.png`): 大脑/AI芯片图标，颜色 `#64748B`
- **选中** (`ai-active.png`): 大脑/AI芯片图标 + 右上角红点提示，颜色 `#F59E0B`
- **设计参考**: 大脑轮廓或电路板图案，active版需带红点(`#EF4444`)表示有新功能

### 5. 我的/个人中心 (profile)

- **默认** (`profile.png`): 人物轮廓图标，颜色 `#64748B`
- **选中** (`profile-active.png`): 人物轮廓图标，颜色 `#F59E0B`
- **设计参考**: 简约人物头像轮廓

## 当前临时方案

在真实图标就绪前，uni-app 的 TabBar 会因找不到图标文件而显示空白。
如需临时测试，可考虑以下替代方案：

1. **使用自定义 TabBar 组件**: 用文字/unicode 符号代替图标
2. **使用 base64 内联 SVG**: 将 SVG 直接编码进 pages.json（不推荐，维护困难）
3. **先生成简单的占位PNG**: 使用脚本生成纯色方块作为临时图标
