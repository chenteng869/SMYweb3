# 太初国链官网规划文档

## 一、项目定位

**太初国链官网**是品牌的官方门面，面向三类核心用户：

1. **B端企业客户** — 想了解萨摩亚SPV注册、全球支付、跨境出海服务的企业决策者
2. **C端个人用户** — 想注册成为会员、了解DLC等级体系、使用AI工具的个人用户
3. **生态合作伙伴** — 想加入太初国链生态的物流、支付、法务等服务提供商

官网核心目标：**建立信任 → 展示能力 → 引导转化**

---

## 二、网站架构

### 技术栈建议

- **前端框架**: Next.js 14 (SSR/SSG，SEO友好) 或 React + Vite (SPA)
- **样式方案**: Tailwind CSS + shadcn/ui
- **动画**: Framer Motion + GSAP (滚动动画)
- **部署**: 静态站点部署 (CDN加速)
- **多语言**: 中/英/日/韩 (第一阶段先做中英)

### 页面结构 (共10页)

```
/
├── 首页 (Home)
├── 萨摩亚SPV (SamoaSPV)
│   └── 子页: 注册流程 / 定价方案 / 常见问题
├── 全球支付 (GlobalPayments)
│   └── 子页: 支付通道 / 费率说明 / 接入指南
├── AI大脑 (AIBrain)
│   └── 子页: 智能体介绍 / 使用场景 / API文档
├── 会员体系 (Membership)
│   └── 子页: DLC等级 / DVSF分红 / 权益对比
├── 资源中心 (Resources)
│   ├── 博客文章 (Blog)
│   ├── 视频教程 (Videos)
│   ├── 白皮书 (Whitepapers)
│   └── 合规文档 (Compliance)
├── 关于我们 (About)
│   └── 子页: 团队介绍 / 合作伙伴 / 新闻动态
├── 联系我们 (Contact)
└── H5 App入口 (AppDownload)
```

## 三、各页面详细设计

### 1. 首页 (Home) — 最重要

**设计目标**: 3秒内传达核心价值，10秒内建立信任

**板块设计**:

| 板块           | 内容                                                             | 视觉风格                               |
| -------------- | ---------------------------------------------------------------- | -------------------------------------- |
| **Hero**       | 大标题"全球出海，从太初开始" + 副标题 + CTA双按钮                | 全屏视频背景或动态粒子效果，深色科技感 |
| **信任背书**   | 4个数据: 服务300+企业 / 覆盖150+国家 / 年处理$2B+ / 7×24智能服务 | 数字滚动动画                           |
| **核心服务**   | 3张卡片: 萨摩亚SPV / 全球支付 / AI大脑                           | 悬浮卡片+微交互动画                    |
| **萨摩亚优势** | 地图+8个优势点: 零税/隐私/英联邦/无外汇管制等                    | 交互式世界地图                         |
| **AI能力展示** | 10个AI智能体图标网格 + 实时数据                                  | 动态网格布局                           |
| **客户案例**   | 3-5个真实客户logo + 简短评价                                     | 轮播或横向滚动                         |
| **会员等级**   | DLC 5级体系可视化                                                | 阶梯式上升动画                         |
| **CTA**        | "立即开启全球出海之旅" + 注册按钮                                | 全屏渐变背景                           |
| **Footer**     | 多列链接 + 社交媒体 + 备案信息                                   | 简洁深色                               |

**关键CTA按钮**:

- 主按钮: "免费注册" (金色渐变)
- 次按钮: "预约咨询" (边框按钮)
- H5入口: "下载App" (带二维码)

### 2. 萨摩亚SPV页面 (SamoaSPV)

**目标**: 说服用户选择萨摩亚作为出海基地

**板块**:

1. **Hero**: 萨摩亚海岛全景 + 核心卖点"零税率 · 隐私保护 · 全球通行"

7. **CTA**: "开始注册"按钮

## 四、设计规范

### 色彩系统

| 用途     | 色值    | 说明                |
| -------- | ------- | ------------------- |
| 主背景   | #0A0E1A | 深海蓝黑            |
| 次背景   | #111827 | 卡片背景            |
| 强调色   | #F6A623 | 珊瑚金，CTA按钮     |
| 辅助色   | #00D4AA | 太平洋青，成功/数据 |
| 警示色   | #CE1126 | 萨摩亚红，紧急/重要 |
| 文字主色 | #F9FAFB | 纯白偏暖            |
| 文字次色 | #9CA3AF | 灰色                |

### 字体规范

- **英文标题**: Space Grotesk (几何感)
- **中文标题**: 系统默认 + font-weight: 700
- **正文**: Inter + 系统字体栈
- **数据**: JetBrains Mono (等宽)

### 动画规范

- **页面滚动**: GSAP ScrollTrigger 驱动
- **元素入场**: Framer Motion (fade + translateY)
- **数据展示**: count-up 数字滚动
- ** hover效果**: 缩放0.98 + 颜色过渡
