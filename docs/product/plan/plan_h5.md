# H5 模拟版本开发计划

基于白皮书 V7.0 全部22个章节，制作完整的H5模拟版本。

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS 3.4
- Framer Motion (动画)
- Zustand (状态管理)
- React Router (路由)
- Lucide React (图标)
- Recharts (图表)
