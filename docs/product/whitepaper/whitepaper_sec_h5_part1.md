# 3.2 H5 模拟版本设计方案

## 3.2.1 图片素材选择方案

> **设计原则**：太初国链 H5 的视觉体系遵循「海岛主权 × 科技金融」双重基因——以南太平洋萨摩亚的自然景观与主权符号构建信任锚点，以 AI 粒子特效与数字资产图形传递技术领先感。所有图片素材按「核心视觉」「科技 AI」「商业金融」「数字资产」四大类目分级管理，确保首屏加载 < 1.5s，全站图片总预算 < 8MB（移动端）。

#### 5.2 图片加载分级策略（性能优化）

```javascript
/**
 * 图片加载优先级策略
 * 目标：首屏 FCP < 1.2s，LCP < 2.5s
 */
const ImageLoadingStrategy = {
  // P0: 首屏可见，立即加载（Preload）
  critical: [
    'V-01',   // Hero 背景
    'V-02',   // 国旗图标
    'AI-01',  // AI 大脑页头
    'AI-05',  // 加载动画
  ],

  // P1: 首屏下方 1 屏内，IntersectionObserver 懒加载
  important: [
    'V-03',   // 刺绣徽章
    'AI-02',  // 芯片大脑
    'B-01',   // 注册流程
    'D-01',   // 6类资产对比
  ],

  // P2: 深层页面素材，路由级按需加载
  deferred: [
    'V-04', 'V-05', 'V-06',
    'AI-03', 'AI-04', 'AI-06',
    'B-02', 'B-03', 'B-04', 'B-05', 'B-06', 'B-07', 'B-08',
    'D-02', 'D-03', 'D-04', 'D-05',
  ],
};

// 技术实现：图片 CDN + 自适应格式
const imageConfig = {
  baseURL: 'https://img.taichuglobal.com',
  formats: ['avif', 'webp', 'jpg'],  // 按支持度降级
  breakpoints: [375, 750, 1125],      // 1x, 2x, 3x DPR
  quality: { avif: 70, webp: 75, jpg: 80 },
};
```

#### 5.3 素材交付标准

| 维度 | 标准要求 |
|-----|---------|
| 分辨率 | 所有位图素材提供 3x 版本（1125px 宽），矢量素材提供 SVG |
| 色彩空间 | sRGB IEC61966-2.1，导出时嵌入 ICC 配置文件 |
| 透明素材 | PNG-24 或 WebP（有损+透明），禁用 GIF 透明 |
| 动效素材 | Lottie JSON 优先（< 100KB），次选 GIF-8bit（< 200KB），末选视频 |
| 命名规范 | `{类目}-{序号}_{用途}_{尺寸}.{格式}`，如 `V-01_Hero-Background_375x812.jpg` |
| 版权状态 | 所有素材附带授权证书，定制素材签署完整著作权转让协议 |

## 3.2.2 H5 整体架构

> **架构理念**：太初国链 H5 采用「单页应用（SPA）+ 底部 Tab 导航」的经典移动端架构，以 React 18 + Vite 为技术底座，Zustand 管理全局状态，React Query 处理服务端状态。整体遵循「移动端优先（Mobile First）」设计策略，所有页面以 375px 逻辑像素为基准设计，通过弹性布局适配 320px~428px 全段位移动设备。

#### 1.3 Tab 切换交互规范

| 交互类型 | 行为定义 | 技术实现 |
|---------|---------|---------|
| 点击切换 | 点击 Tab → 目标页面从右向左滑入（300ms ease-out） | Framer Motion `AnimatePresence` + `initial={{x:100}}` |
| 手势返回 | 从屏幕左边缘向右滑动 → 返回上一页 | `touchstart` + `touchmove` 事件监听，位移 > 80px 触发 |
| Tab 再点击 | 已激活 Tab 再次点击 → 页面回滚到顶部 | `window.scrollTo({top:0, behavior:'smooth'})` |
| AI Tab 特殊态 | AI 有新建议时 → 图标持续播放脉冲呼吸动画 | CSS `@keyframes pulse {0%{scale:1}50%{scale:1.12}100%{scale:1}}` 2s infinite |
| 角标更新 | 全局 WebSocket 监听消息 → 角标数字实时更新 | Zustand `useNotificationStore` 订阅 WS 事件 |

// 动态根字体计算 (rem 适配方案)
function setRem() {
  const designWidth = 375;  // 设计稿基准宽度
  const baseFontSize = 16;  // 基准字体 16px
  const clientWidth = document.documentElement.clientWidth;
  const scale = clientWidth / designWidth;
  const fontSize = Math.min(scale * baseFontSize, 20); // 上限 20px 防过大
  document.documentElement.style.fontSize = `${fontSize}px`;
}

// 安全区适配 (iPhone 刘海屏 / 底部指示条)
const safeAreaConfig = {

};

// viewport 配置

#### 5.4 AI 排序算法

```typescript
/**
 * AI 待办优先级评分算法
 * 综合多因子计算 priority_score，按分数降序排列
 */
interface TodoItem {
  id: string;
  title: string;
  category: 'compliance' | 'tax' | 'legal' | 'trade' | 'banking';
  deadline?: Date;
  aiPrefilled: boolean;
  financialImpact: number;    // 资金影响金额 ($)
  complianceRisk: number;     // 合规风险评分 (0-100)
}

function calculatePriorityScore(todo: TodoItem): number {
  const now = Date.now();
  const daysUntilDeadline = todo.deadline
    ? (todo.deadline.getTime() - now) / (1000 * 60 * 60 * 24)
    : 999;

  // 时间紧迫度分数 (0-100)，越接近截止日期分数越高
  const urgencyScore = Math.max(0, 100 - (daysUntilDeadline * 14));

  // 资金影响分数 (0-100)，按对数缩放
  const financialScore = Math.min(100, Math.log10(todo.financialImpact + 1) * 20);

  // 合规风险分数 (0-100)
  const riskScore = todo.complianceRisk;

  // AI 预填加成 (已预填的待办更容易完成，优先展示)
  const aiBonus = todo.aiPrefilled ? 15 : 0;

  // 权重配置
  const weights = {
    urgency: 0.35,      // 时间紧迫度 35%
    financial: 0.25,    // 资金影响 25%
    compliance: 0.30,   // 合规风险 30%
    aiBonus: 0.10,      // AI 加成 10%
  };

  return (
    urgencyScore * weights.urgency +
    financialScore * weights.financial +
    riskScore * weights.compliance +
    aiBonus * weights.aiBonus
  );
}

// 排序后的待办列表
const sortedTodos = todos
  .map(t => ({ ...t, priorityScore: calculatePriorityScore(t) }))
  .sort((a, b) => b.priorityScore - a.priorityScore);
```
