### 10.2 关键联动逻辑

#### 联动 1: DLC等级 → AI Agent功能解锁

```
触发条件: 用户DVC达到等级阈值
联动行为:
  1. 系统推送升级通知 (Push + 站内消息)
  2. AI大脑页对应Agent卡片解锁 (灰色→彩色)
  3. Agent卡片显示"NEW"角标，持续7天
  4. 首页Banner显示升级提醒
  5. 分红权重自动更新

降级保护: DVC减少不会导致等级下降
          等级一旦解锁永久保留
```

#### 联动 2: 公司年审 → 全局提醒

```
触发条件: 任意公司年审倒计时<30天
联动行为:
  1. 会员中心"我的公司"显示红点
  2. 公司卡片边框变红
  3. 订单追踪页显示维护订单
  4. 推送通知 (按倒计时规则)
  5. AI合规官发出预警
  6. 首页显示顶部警告Banner

解除条件: 年审完成后自动解除所有提醒
```

#### 联动 3: DVSF分红 → 资产更新

```
触发条件: 每月1日 00:00 UTC
联动行为:
  1. 系统自动计算分红
  2. DVSF分红池页显示"已发放"状态
  3. 全球收款控制台余额更新
```
