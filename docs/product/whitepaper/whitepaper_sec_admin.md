# 8.5 新增后台独立页面线框图

> 本章为太初国链管理员后台（Admin Dashboard）新增独立页面提供完整的UI线框设计、组件规格与交互定义。所有线框基于 **React 18 + Ant Design Pro v5 + 深色模式（Dark Theme）** 绘制，色彩变量遵循 Ant Design 官方暗色令牌（`colorBgBase: #141414`、`colorBgContainer: #1f1f1f`、`colorBorder: #434343`）。

> **文档版本**: v1.0 | **适用产品版本**: 太初国链 Admin Dashboard v2.5+ | **技术栈**: React 18 + Ant Design Pro v5 + TypeScript + Recharts + WebSocket

## 8.5.1 公司注册订单管理页

### 页面概览

公司注册订单管理页是后台运营核心模块，承载全球公司注册订单的全生命周期管理。页面采用 **筛选栏 → 统计卡片区 → 批量操作栏 → 表格主体** 的四层纵向布局，支持多维度筛选、状态聚合与批量操作。面向运营团队、代理人主管和合规审核员三类角色，通过权限控制展示差异化操作。

### 表格列定义（TypeScript）

```typescript
interface CompanyOrderColumn {
  orderNo: string;              // 订单号，格式 REG-{YY}{NNNNN}
  companyEn: string;            // 公司英文名
  companyCn: string;            // 公司中文名
  applicant: string;            // 申请人姓名
  applicantId: string;          // 申请人用户ID
  applicantAvatar?: string;     // 申请人头像
  status: 'pending' | 'reviewing' | 'registered' | 'supplement' | 'rejected' | 'expiring';
  submitTime: string;           // ISO 8601
  agentName: string;            // 代理人名称
  agentId: string;
  agentAvatar?: string;
  jurisdiction: string;         // 注册地: HK / SG / US / UK / WS
  companyType: string;
  reviewerNotes?: string;
  attachments: Attachment[];
  timeline: TimelineEvent[];
}

interface Attachment {
  id: string;
  name: string;
  type: string;
  size: number;
  status: 'uploaded' | 'missing' | 'pending_review';
  url: string;
  uploadedAt: string;
}

interface TimelineEvent {
  id: string;
  status: string;
  operator: string;
  timestamp: string;
  note?: string;
}

const columns: ProColumns<CompanyOrderColumn>[] = [
  {
    title: '订单号',
    dataIndex: 'orderNo',
    width: 130,
    fixed: 'left',
    sorter: true,
    render: (text) => <Tag color="blue" style={{ fontFamily: 'monospace' }}>{text}</Tag>,
  },
  {
    title: '公司名',
    dataIndex: 'companyEn',
    width: 260,
    ellipsis: true,
    render: (_, record) => (
      <div>
        <div style={{ fontWeight: 600, fontSize: 14 }}>{record.companyEn}</div>
        {record.companyCn && (
          <div style={{ fontSize: 12, color: '#8C8C8C', marginTop: 2 }}>{record.companyCn}</div>
        )}
        <div style={{ marginTop: 4 }}>
          <Tag size="small">{getJurisdictionFlag(record.jurisdiction)} {record.jurisdiction}</Tag>
        </div>
      </div>
    ),
  },
  {
    title: '申请人',
    dataIndex: 'applicant',
    width: 120,
    render: (text, record) => (
      <Space><Avatar size="small" src={record.applicantAvatar} /><span>{text}</span></Space>
    ),
  },
  {
    title: '状态',
    dataIndex: 'status',
    width: 110,
    filters: true,
    onFilter: true,
    valueEnum: {
      pending: { text: '待审核', status: 'Warning' },
      reviewing: { text: '审核中', status: 'Processing' },
      registered: { text: '已注册', status: 'Success' },
      supplement: { text: '需补件', status: 'Error' },
      rejected: { text: '已驳回', status: 'Default' },

  {
    title: '提交时间',

    width: 160,

      <Tooltip title={dayjs(text).format('YYYY-MM-DD HH:mm:ss')}>

      </Tooltip>
    ),
  },
  {
    title: '代理人',

    ),
  },
  {
    title: '操作',
    valueType: 'option',
    width: 220,
    fixed: 'right',
    render: (_, record) => (
      <Space size="small">
        <Button key="view" size="small" icon={<EyeOutlined />} onClick={() => showDetailDrawer(record)}>查看</Button>
```

### 状态标签颜色映射（深色模式）

| 状态值 | 徽章类型 | 背景色 | 文字色 | 触发条件 | 对应操作 |

| `pending` | `Warning` | `#2B1D11` | `#D46B08` | 用户提交资料后 | 审核/催办 |
| `reviewing` | `Processing` | `#1D2B2B` | `#5BD4D0` | 代理人开始审核 | 查看/转交 |
| `registered` | `Success` | `#1B2E1B` | `#5BD46E` | 政府完成注册 | 查看/年审设置 |

| `rejected` | `Default` | `#262626` | `#8C8C8C` | 严重不合规 | 归档/重新申请 |
| `expiring` | `Warning` | `#2B1D11` | `#D46B08` | 年审即将到期 | 年审提醒 |

### API 接口

```typescript
// 获取订单列表
GET /api/admin/company-orders
Params: {
  status?: 'pending' | 'reviewing' | 'registered' | 'supplement' | 'rejected' | 'expiring';
  submitDateStart?: string; submitDateEnd?: string;
  agentId?: string; companyName?: string; jurisdiction?: string;
  page?: number; pageSize?: number; sortField?: string; sortOrder?: 'ascend' | 'descend';
}
Response: { data: CompanyOrder[]; total: number;
  summary: { pending: number; reviewing: number; registered: number; expiring: number; };
}

// 批量操作
POST /api/admin/company-orders/batch
Body: { action: 'export' | 'assign_agent' | 'send_reminder'; ids: string[]; params: Record<string, any>; }

// 审核订单
POST /api/admin/company-orders/:id/audit
Body: { action: 'approve' | 'reject' | 'request_supplement'; notes: string; attachments?: string[]; }

// 发送催办
POST /api/admin/company-orders/:id/remind
Body: { channels: ('email' | 'sms' | 'app')[]; message?: string; }
```

## 8.5.7 DLC等级配置页

### 页面概览

DLC（Decentralized Level Certificate）等级配置页是平台经济系统的核心参数面板，负责定义L1-L9共9个等级的全部配置参数。页面采用 **横向标签切换等级 → 纵向配置面板** 的布局，每个等级包含等级标识、升级条件、特权开关和权重系数四大配置区块。L9为预留荣誉等级，需完成特殊成就方可解锁。

```typescript
type DLCLevel = 'L1' | 'L2' | 'L3' | 'L4' | 'L5' | 'L6' | 'L7' | 'L8' | 'L9';

  icon: string;
  themeColor: string;
  textColor: string;
  description: string;
  badgeImageUrl?: string;
  isEnabled: boolean;
  isHonorLevel: boolean;
  displayOrder: number;
  upgradeConditions: UpgradeCondition[];
  conditionLogic: 'all' | 'any' | 'weighted';
  minWeightedScore?: number;
  privileges: PrivilegeConfig[];

  userCount: number;
  dvcThreshold: number;
}

type ConditionType = 'dvc_balance' | 'total_trade_amount' | 'content_count' | 'follower_count'
  | 'consecutive_active_days' | 'invite_count' | 'staking_amount' | 'community_score';

interface UpgradeCondition {
  id: string;
  type: ConditionType;
  operator: '>=' | '>' | '=' | '<=';
  threshold: number;
  unit: string;
  isRequired: boolean;
  weight: number;
}

interface PrivilegeConfig {
  id: string;
  name: string;

  icon: string;
  isEnabled: boolean;

}
```
