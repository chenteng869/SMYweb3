# 3.2.8 税务计算器页（Tax Optimizer）

## 功能定位

税务计算器是太初国链平台的核心AI决策工具，旨在帮助跨境电商卖家在**60秒内**量化评估不同公司架构（纯内地 / 海南 / 海南+萨摩亚SPV / 纯离岸）的税务成本差异。该模块依托太初AI超级大脑的全球税务知识库，将原本需要数周、花费数万美金的税务咨询压缩为一次点击的实时计算。

**核心价值主张**：

| 维度     | 传统方案                  | 太初国链方案             |
| -------- | ------------------------- | ------------------------ |
| 时间成本 | 2-4周（预约→面谈→出报告） | 60秒实时计算             |
| 金钱成本 | $3,000-$15,000/次咨询     | 平台会员免费             |
| 覆盖范围 | 单一国家/地区             | 20+国家/地区同步对比     |
| 更新频率 | 年度更新                  | 实时汇率+法规联动        |
| 可视化   | PDF静态报告               | 动态柱状图+路径图+AI建议 |

**目标用户画像**：

| 用户类型   | 年营收区间  | 典型场景                     | 核心诉求              |
| ---------- | ----------- | ---------------------------- | --------------------- |
| 初创卖家   | $100K-$500K | 刚起步，考虑是否注册海南公司 | 验证架构成本优势      |
| 成长型卖家 | $500K-$2M   | 多市场布局，税务复杂度上升   | 寻找最优税负组合      |
| 成熟卖家   | $2M-$10M    | 已有架构，考虑SPV升级        | 精确计算升级ROI       |
| 企业级卖家 | $10M+       | 全球布局，需专业税务规划     | 获取完整方案+执行路径 |

```
用户操作                          系统响应
─────────────────────────────────────────────────────────
页面加载           →   表单从底部滑入，字段显示默认值
                   →   AI知识库状态检测，显示"已加载23国"

点击快捷按钮       →   营收字段即时更新对应数值
                   →   按钮切换激活态样式

修改毛利率         →   进度条实时跟随更新
                   →   验证范围 5%-95%

点击"开始计算"     →   按钮进入加载态（spinner旋转）
                   →   前端本地计算 + 调用AI验证API
                   →   800ms后结果区以动画展开
                   →   柱状图逐条生长动画（每条200ms间隔）

计算异常           →   显示错误提示 Toast
                   →   "计算服务繁忙，请稍后重试"
```

# 3.2.9 法务中台页（Legal Hub）

## 功能定位

法务中台是太初国链平台面向跨境电商卖家的**一站式AI法务解决方案**，覆盖从市场准入合规、合同自动生成、制裁风险筛查到知识产权保护的全链路法务需求。通过与太初AI超级大脑的全球法规知识库深度联动，将原本需要多个专业律师团队（国际商事律师、合规顾问、知识产权律师）协作数周才能完成的法务工作，压缩为几次点击的智能化操作。

**核心价值主张**：

| 维度     | 传统方案                    | 太初国链方案       |
| -------- | --------------------------- | ------------------ |

| 合同起草 | $2,000-$10,000/份，1-2周    | AI生成，30秒/份    |
| 制裁筛查 | 订阅专业数据库，$15,000+/年 | 平台内置，实时筛查 |
| 法规更新 | 人工跟踪，滞后数月          | AI爬虫实时监测     |
| 语言支持 | 需单独翻译                  | 内置50+语言生成    |

| 用户类型       | 典型场景             | 核心诉求               |
