# 3.2.4 发现页 · 全球机遇（详细UI线框）

## 页面概述

发现页是用户了解萨摩亚离岸优势、全球支付生态和太初国链服务能力的核心入口。采用PPT沉浸式滑动叙事 + 卡片式功能入口的复合布局，兼顾教育性和转化性。

| 属性 | 规格 |
|------|------|
| 页面标题 | 发现 · 全球机遇 |
| 路由 | `/discover` |
| 布局结构 | 顶部Hero轮播 + 中部网格卡片 + 底部生态入口 |
| 垂直滚动 | 支持（总高度约2400px） |
| 背景色 | 深海蓝渐变 `#0a1628` → `#0d1f3c` |
| 主导航 | 底部Tab Bar（发现页高亮） |

## 3.2.4.1 萨摩亚国家介绍页（PPT式滑动设计）

### 设计说明

采用全屏横向滑动（Horizontal Swipe）的PPT式叙事体验，共8个章节卡片。每个章节占满屏幕宽度，用户可左右滑动切换，底部显示章节指示器。纵向内容超出屏幕时可上下滚动阅读。

| 属性 | 规格 |
|------|------|
| 容器尺寸 | 375px × 100%（全屏宽） |
| 章节数量 | 8个 |
| 滑动方式 | 横向手势滑动 + 底部指示器点击跳转 |
| 自动播放 | 关闭（尊重用户阅读节奏） |
| 过渡动画 | 300ms ease-out 水平位移 |
| 循环滑动 | 否（到首尾边界弹性回弹） |
