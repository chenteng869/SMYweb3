### 1.2 登录流程

#### 正常登录

1. 在「用户名」输入框输入您的账号（邮箱或手机号）
2. 在「密码」输入框输入您的登录密码
   - ⚠️ 密码区分大小写
   - ⚠️ 注意切换中英文输入法
3. 如果出现验证码，请输入图片中的字符（不区分大小写）
4. 点击「登录」按钮或按 Enter 键

**首次登录提示**:

- 首次登录后系统会强制要求您修改初始密码
- 新密码要求：至少12位，包含大小写字母+数字+特殊字符
- 修改成功后将自动跳转到主界面

#### 验证码机制

| 场景            | 是否需要验证码 | 说明                    |
| --------------- | -------------- | ----------------------- |
| 正常登录        | 否             | 直接登录                |
| 连续失败1-2次   | 否             | 仅提示错误              |
| 连续失败≥3次    | ✅ 是          | 显示图形验证码          |
| 连续失败≥5次    | ✅ 是          | 切换为滑动验证/短信验证 |
| 异地登录/IP异常 | ✅ 是          | 触发增强安全验证        |

### 1.3 安全注意事项

**密码安全**:

- 🔒 请勿使用与其它网站相同的密码
- 🔒 定期更换密码（建议每90天）
- 🔒 不要将密码写在便签或共享文档中
- 🔒 使用密码管理器（如 1Password, Bitwarden）

**会话安全**:

- 🛡️ 公共电脑使用后务必点击「退出登录」
- 🛡️ 不要在浏览器中保存密码（公共环境）
- 🛡️ 如发现异常操作，立即修改密码并联系管理员
- 🛡️ 开启两步验证（2FA）可大幅提升账户安全性

**网络环境要求**:

- ✅ 推荐使用公司内网或可信Wi-Fi
- ⚠️ 公共Wi-Fi环境下建议使用VPN
- ❌ 禁止在不安全的网络环境中进行敏感操作

## 2. DID身份管理

### 2.1 注册新DID身份

**入口**: 左侧菜单 → 「DID身份管理」→ 点击「+ 新建DID」

**操作步骤**:

1. **填写基本信息**
   - 名称（必填）：给这个DID身份起一个便于识别的名称
     - 示例：「张三的个人DID」、「SMYWeb3运营部官方身份」
     - 长度限制：2-50个字符
   - 类型选择（必填）：
     - `个人` - 用于个人业务场景
     - `组织` - 用于企业/团队场景
   - 描述信息（选填）：补充说明该身份的用途

2. **确认创建**
   - 点击「提交创建」按钮
   - 系统显示确认弹窗：
     ```
     ┌─────────────────────────────────────┐
     │  ⚡ 即将创建新的DID身份              │
     │                                     │
     │  名称: 我的个人DID                   │
     │  类型: 个人                         │
     │                                     │
     │  创建后将生成链上NFT代币             │
     │  此操作不可撤销                      │
     │                                     │
     │  [ 取消 ]  [ 确认创建 ]              │
     └─────────────────────────────────────┘
     ```

3. **等待链上确认**
   - 创建成功后，状态显示为「待确认」(pending)
   - 页面顶部显示进度条和预计等待时间
   - 通常需要30秒至3分钟完成链上确认
   - 确认完成后自动刷新为「活跃」(active)状态

**💡 小贴士**:

- 一个账号最多可创建20个DID身份
- 组织类型DID支持添加多个管理员
- 建议为不同业务场景创建独立的DID身份

### 2.3 身份状态管理

**状态流转图**:

```
pending (待确认)
    │
    │ 链上确认成功
    ▼
active (活跃) ←→ frozen (冻结) ←→ unfreezing (解冻中)
    │                                        │
    │ 申请吊销                               │ 解冻成功
    ▼                                        ▼
revoking (吊销中) ←←←←←←←←←←←←←←←←←←←← revoked (已吊销)
    │
    │ 链上确认吊销
    ▼
  (不可逆，仅保留历史记录)
```

**各状态下允许的操作**:

| 操作     | pending | active | frozen | revoked |
| -------- | ------- | ------ | ------ | ------- |
| 查看详情 | ✅      | ✅     | ✅     | ✅      |
| 编辑信息 | ❌      | ✅     | ❌     | ❌      |
| 用于存证 | ❌      | ✅     | ❌     | ❌      |
| 用于签名 | ❌      | ✅     | ❌     | ❌      |
| 冻结     | ❌      | ✅     | ❌     | ❌      |
| 解冻     | ❌      | ❌     | ✅     | ❌      |
| 吊销     | ❌      | ✅     | ✅     | ❌      |

**⚠️ 重要提醒**:

- 吊销操作不可逆！请谨慎执行
- 冻结的身份无法用于新存证和签名，但历史记录保留
- 已有存证/签名的DID被吊销后，相关记录仍可查询但标记为"已失效"

## 3. KYC认证管理

### 3.1 提交认证材料

**入口**: 左侧菜单 → 「KYC认证」→ 「申请新认证」

**所需材料清单**:

| 材料项       | 要求     | 格式           | 说明                          |
| ------------ | -------- | -------------- | ----------------------------- |
| 身份证件照   | 必填     | JPG/PNG, ≤5MB  | 身份证正反面或护照首页        |
| 人脸活体检测 | 必填     | 实时拍摄       | 系统引导完成                  |
| 手持证件照   | 必填     | JPG/PNG, ≤5MB  | 需同时露出人脸和证件          |
| 地址证明     | 选填     | PDF/JPG, ≤10MB | 近3个月的水电账单或银行对账单 |
| 营业执照     | 组织必填 | PDF, ≤10MB     | 企业需提供                    |

**操作流程**:

1. 选择要关联的DID身份（下拉选择已激活的DID）

   - ⚠️ 请在光线充足的环境下进行
4. 上传手持证件自拍
5. （可选）上传地址证明

## 4. 区块链存证

### 4.1 上传文件并存证

**入口**: 左侧菜单 → 「区块链存证」→ 「新建存证」

**支持的文件类型**:

| 类别      | 支持格式                             | 最大大小 |
| --------- | ------------------------------------ | -------- |
| 文档      | PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX | 100MB    |
| 图片      | JPG, PNG, GIF, WEBP, BMP, TIFF       | 50MB     |
| 音频      | MP3, WAV, AAC, FLAC                  | 200MB    |
| 视频      | MP4, MOV, AVI, MKV                   | 500MB    |
| 代码/文本 | ZIP, RAR, 7Z, TXT, JSON, XML         | 100MB    |
| 其他      | 任意二进制文件                       | 500MB    |

**操作步骤**:

1. **上传文件**
   - 拖拽文件到虚线框区域，或点击「选择文件」
   - 支持多文件批量上传（单次最多20个）
   - 上传过程中显示进度条和速度

2. **填写存证信息**

   ```
   标题*: [________________]
   例: 《2026年Q2供应商合作协议》

   描述: [________________________________]
   例: 与XX公司的年度采购框架协议

   分类*: [ 选择分类 ▼ ]
   ▶ 合同协议
   ▶ 知识产权
   ▶ 财务凭证
   ▶ 人事档案
   ▶ 其他

   关联DID: [ 选择DID身份 ▼ ]
   (可选，关联后可在该DID下统一查看)

   标签: [+ 添加标签]
   [重要] [合同] [Q2] [×] ...

   ☑️ 公开此存证 (生成公开验证链接)
   ```

3. **预览并确认**
   - 系统自动计算文件哈希值（SHA-256）

## 5. 电子签名

### 5.1 选择文档进行签名

**前置条件**:

- 文档必须已完成区块链存证
- 签署者必须有有效的DID身份
- 签署者账户已完成KYC认证

**操作路径**:

1. 进入「区块链存证」模块
2. 找到需要签署的文档
3. 点击操作列的「发起签名」按钮

或从「电子签名」模块入口：

1. 点击「新建签名请求」
2. 在弹出的文档选择器中选择已存证的文档

### 5.2 选择签名算法

系统支持以下签名算法：

| 算法              | 安全等级 | 适用场景            | 国密合规        | 签名速度 |
| ----------------- | -------- | ------------------- | --------------- | -------- |
| **SM2**           | ★★★★★    | 国内业务首选        | ✅ 符合国密标准 | 快       |
| ECDSA (secp256k1) | ★★★★☆    | 国际业务/以太坊生态 | ❌              | 快       |
| Ed25519           | ★★★★☆    | 高性能场景          | ❌              | 最快     |

**选择建议**:

- 中国大陆境内业务 → **SM2**（符合《密码法》要求）
- 跨境业务 → **ECDSA**（国际通用）
- 高频批量签名 → **Ed25519**

   │  ☑️ 我同意对此文档进行电子签名            │

## 附录：常见问题 FAQ

**Q1: 忘记密码怎么办？**
A: 点击登录页的「忘记密码」，输入注册邮箱，系统会发送重置链接。链接24小时内有效。

**Q2: 界面显示异常/白屏？**
A: 尝试以下步骤：① 清除浏览器缓存 ② 使用Ctrl+F5强制刷新 ③ 更换Chrome浏览器 ④ 联系技术支持。

**Q3: 上传文件失败？**
A: 检查：① 文件大小是否超限 ② 文件格式是否支持 ③ 网络是否稳定 ④ 浏览器控制台是否有报错。

**Q4: 数据不同步？**
A: 检查平台适配器的OAuth授权是否过期（Instagram每60天需刷新），或手动触发一次同步。

**Q5: 如何联系技术支持？**
A:

- 一般问题: help@smyweb3.com
- 紧急问题: 在系统右下角点击「在线客服」
- 电话支持: 400-XXX-XXXX (工作时间 9:00-18:00)
