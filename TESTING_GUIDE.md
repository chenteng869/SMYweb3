# 国学出海 Web3 Dapp - 测试指南

## 测试概述

本指南帮助您验证和测试管理后台系统的功能完整性和稳定性。

## 🚀 快速启动与验证

### 1. 环境准备
```bash
# 检查 Node.js 版本
node --version  # 需要 18+

# 检查 npm 版本
npm --version
```

### 2. 前端启动与测试
```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问: http://localhost:3000
```

### 3. 后端启动与测试
```bash
cd backend

# 安装依赖
npm install

# 启动开发服务器
npm run start:dev

# 访问 API: http://localhost:3001
# 访问 Swagger文档: http://localhost:3001/api/docs
```

## 🐛 常见问题排查

### 前端问题

#### 问题：端口 3000 被占用
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <进程ID> /F

# 或修改端口
# 在 package.json 中修改 dev 脚本
# "dev": "next dev -p 3001"
```

#### 问题：依赖安装失败
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

#### 问题：页面显示 404
- 检查 Next.js 路由是否正确
- 确认页面文件在正确的位置
- 查看浏览器控制台错误信息

### 后端问题

#### 问题：数据库连接失败
- 检查 PostgreSQL 是否启动
- 确认 .env 文件中 DATABASE_URL 配置正确
- 确认数据库用户权限正确

#### 问题：Prisma 类型未找到
```bash
# 重新生成 Prisma Client
npx prisma generate
```

#### 问题：API 请求失败
- 检查后端服务是否正常启动
- 查看浏览器网络请求
- 检查 API 地址配置
- 查看后端日志
